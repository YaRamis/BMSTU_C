Название
Масса
Объем

- 02 - префикс "ALL";
- 03 - префикс "aLl";

- 06 - в файле только одна структура;

- 01 - нету аргументов;

- 06 - названий начинающихся с данного префикса нету;
- 07 - длина названия > 25;

- 09 - объем равен нулю;
